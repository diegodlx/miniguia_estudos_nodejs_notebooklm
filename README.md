🚀 Especialização Backend: JavaScript & Node.js com NotebookLM

📌 Contexto

Este repositório documenta minha jornada de estudos utilizando o NotebookLM como assistente de aprendizagem.

O material base para as interações e análises são as documentações oficiais da MDN Web Docs e do Node.js, garantindo uma base teórica sólida, confiável e atualizada.

O objetivo deste ambiente é evoluir meu conhecimento técnico de iniciante → confortável, permitindo uma transição consistente do desenvolvimento frontend para a construção de sistemas robustos no backend.

🎯 O Assunto: Node.js

O Node.js é um ambiente de execução JavaScript open-source, cross-platform e altamente performático.

Ele utiliza o motor V8 (Google Chrome) para executar código fora do navegador, possibilitando o uso da mesma linguagem tanto no cliente quanto no servidor.
