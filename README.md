# 📘 Miniguia de Estudos Node.js com NotebookLM

## 🚀 Sobre o Projeto

Este repositório documenta minha jornada de especialização em **Backend
com JavaScript e Node.js**, utilizando o **NotebookLM** como assistente
de aprendizado.

A proposta é transformar documentação técnica em conhecimento prático,
estruturado e aplicável no dia a dia de um desenvolvedor backend.

📌 Base de estudo: - MDN Web Docs\
- Documentação oficial do Node.js

# 🧠 Prompts Reutilizáveis de Estudo

> 💡 **Dica:** tente responder antes de consultar os resumos.

## 📌 1. Fundamentos e JavaScript Core

### Herança

Explique como funciona a cadeia de protótipos (prototype chain) no
JavaScript e como ela se diferencia da herança em linguagens como Java

### Qualidade de Código

Quais são as principais restrições impostas pelo Modo Estrito (Strict
Mode) e como ele auxilia na performance e depuração de uma aplicação
backend?

### Concorrência

Descreva o funcionamento do Modelo de Concorrência do JavaScript e a
importância do Loop de Eventos para a execução de código assíncrono

------------------------------------------------------------------------

## ⚙️ 2. Arquitetura do Node.js

### Performance

Como o Node.js utiliza o motor V8 para executar código fora do navegador
e por que isso o torna performático para aplicações de larga escala?

### I/O Não-Bloqueante

Explique o conceito de I/O não-bloqueante no Node.js. Como um único
processo consegue lidar com milhares de conexões simultâneas sem criar
novas threads para cada requisição?

### Ciclo de Vida

Qual a diferença prática entre as funções process.nextTick() e
setImmediate() dentro do ciclo de execução do Node.js?

------------------------------------------------------------------------

## 🧩 3. APIs e Módulos

### Transações HTTP

Descreva a anatomia de uma transação HTTP no Node.js, detalhando o papel
dos objetos http.IncomingMessage (request) e http.ServerResponse
(response)

### Manipulação de Dados

Quando devo preferir o uso de Streams em vez de ler um arquivo inteiro
para a memória usando o módulo fs? Explique o conceito de backpressuring

### Segurança

Quais são as melhores práticas de segurança recomendadas pela
documentação oficial para proteger uma aplicação Node.js em ambiente de
produção?

------------------------------------------------------------------------

## 🧪 4. Testes e Carreira

### Testes Automatizados

Como utilizar o executor de testes nativo do Node.js para criar suítes
de testes, aplicar mocking e verificar a cobertura de código?

### Diagnóstico

Quais ferramentas de diagnóstico o Node.js oferece para analisar
desempenho ruim, vazamentos de memória ou gerar gráficos de chama (flame
graphs)?

### Roadmap

Com base no currículo da MDN, quais são os módulos essenciais que um
desenvolvedor frontend deve dominar para fazer a transição para o
backend com sucesso?
