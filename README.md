Especialização Backend: JavaScript & Node.js com NotebookLM
📌 Contexto
Este repositório documenta minha jornada de estudos utilizando o NotebookLM como assistente de aprendizagem. O material base para as interações e análises são as documentações oficiais da MDN Web Docs e do Node.js, garantindo uma base teórica sólida e atualizada
.
O objetivo deste ambiente é transformar o conhecimento técnico de "iniciante" para "confortável", permitindo uma transição fluida do desenvolvimento frontend para a construção de sistemas robustos no lado do servidor
.
🎯 O Assunto: Node.js
O Node.js é um ambiente de execução JavaScript open-source, cross-platform e altamente performático
. Ele utiliza o motor V8 do Google Chrome para executar código fora do navegador, permitindo que desenvolvedores utilizem a mesma linguagem tanto no cliente quanto no servidor
.
Os pontos centrais do Node.js abordados neste estudo incluem:
Modelo de I/O Não-Bloqueante: Capacidade de lidar com milhares de conexões simultâneas em um único processo, sem desperdiçar ciclos de CPU esperando por respostas de banco de dados ou arquivos
.
Assincronismo: Uso intensivo de Promises, Async/Await e o entendimento profundo do Event Loop
.
Ecossistema: Gerenciamento de dependências via npm e exploração de frameworks como Express
.
🚀 Objetivos de Estudo
O foco principal é a evolução para o cargo de Desenvolvedor Backend, com as seguintes metas claras:
Domínio de JavaScript Core: Compreender conceitos avançados como herança via cadeia de protótipos, Modo Estrito (Strict Mode) para performance e o modelo de concorrência da linguagem
.
Arquitetura de Servidor: Aprender a anatomia de transações HTTP, configuração de redes e criação de servidores web nativos
.
Manipulação de Recursos: Masterizar o trabalho com Sistemas de Arquivos (File System), caminhos (path) e o uso de Streams para transferência eficiente de dados
.
Qualidade de Código: Implementar rotinas de testes automatizados utilizando o executor de testes nativo do Node.js e técnicas de Mocking
.
Segurança e Performance: Aplicar as melhores práticas de segurança e realizar análises de desempenho com gráficos de chama (flame graphs) e diagnóstico de memória
.
🛠️ Fontes Utilizadas
MDN Web Docs: Guias de JavaScript, Web APIs e fundamentos de desenvolvimento web
.
Node.js Documentation: Referência técnica sobre módulos nativos, V8 e padrões de execução
.
